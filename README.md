###مشروع ادراة المواعيد لمادة برمجة التطبيقات الشبكية




##شرح المشروع
يقوم المشروع بادارة المواعيد لكل خدمة مقدمة من كل مقدم خدمة  بحيث يعرض المواعيد المحجوزة ومتابعتها ويجل الزبائن تحجز موعد لاي خدمة بغض النظر عن مقدمها في حال كانت الخدمة متوفرة والموعد المراد حجزه غير محجوز سابقا






##تقسيم العمل :


الطالب حانم جادالله العوابده والطالب مصطفى حسين القاسم نفذوا طلبات المرحلة الاولى واول طلب من المرحلة الثانية


الطالب علي مارديني نفذ الطلب الثاني والثالث من المرحلة الثانية وطلبات المرحلة الثالثة


##كيفية تشغيل المشروع 
يعمل المشروع عن طريق مكتبة الspring boot وادواتها المقدمة من لغة java عن طريق الامر  Run باي بيئة تطوير للغة الjav
